1. Переменная обьявленная с помощью ключевого слова var достпна везде, как внутри блока, так и снаружи.
Также к такой переменной можно обратиться до ее обьявления, просто она будет иметь значение undefined.
2. Переменная обьявленная с помощью ключевого слова let достпна только в пределах блока, в котром обьвлена.
Попытка обратиться к такой переменной до ее обьвления приведет к ошибке. Попытка обьявить в блоке еще одну переменную с таким же именем приведет к ошибке
3. Переменной, обьявленной с помощью ключевого слова const значение задается сразу и изменяться не может.
Попытка изменить значение такой переменной приведет к ошибке.

Использовать переменную var считатеся плохим тоном, потому что такую переменную можно обьвить заново, а значит перезаписать значение, которое уже было присвоено раньше, что может нарушить логику выполнения программы.
